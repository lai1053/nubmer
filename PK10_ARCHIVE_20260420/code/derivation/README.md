# Derivation Code

这个目录收录三条冻结线的上游推理/筛窗代码，以及关键的结果文件。

## 目录说明

- `face/`
  - 双面冻结窗口推理
  - 主要对应 `round35` 与 `round37`
- `sum/`
  - 冠亚和冻结窗口推理
  - 主要对应 `intraday gate`、`intraday bankroll`
- `exact/`
  - 定位胆冻结窗口推理
  - 主要对应日维窗口验证与候选状态缓存

## 说明

- 这里放的是“冻结窗口怎么推出来”的源码与关键结果。
- 当前 live 面板使用的是这些推理结果冻结后的执行版，不会在运行时重新搜索窗口。
